# Custom Mob 📱

A QB-Core admin utility that allows server staff to assign and manage custom premium phone numbers for players through a modern NUI interface.

Designed for FiveM QB-Core servers using `oxmysql`.

## Features

- Custom premium phone number assignment
- Premium number revocation
- Online player selection
- Duplicate phone number detection
- Automatic database synchronization
- SBF Phone System integration
- Permission-based access control
- Modern NUI management panel
- Automatic premium number expiry checks
- Configurable phone number length
- Optional alphanumeric phone numbers
